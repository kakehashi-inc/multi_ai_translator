Multi-AI Translator is a Manifest V3 browser extension (Chrome/Edge/Firefox) that translates web pages or selections via multiple AI providers (OpenAI, Claude, Gemini, Ollama, OpenAI/Anthropic-compatible). It uses Node.js 22+, Vite, ES modules, and vanilla JS/HTML/CSS for popup, options, background, and content scripts. Key folders: src/background (service worker), src/content (page integration), src/popup, src/options, src/providers, src/utils, src/locales; dist/ holds build output, Documents/ has dev/deploy guides.
