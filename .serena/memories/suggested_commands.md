Project uses PowerShell 7 on Windows. Common commands:
- `ls`/`dir`, `cd`, `pwd` for navigation; `cat <file>` or `gc <file>` to read; `clear` to reset terminal.
- `yarn install` to install deps.
- `yarn dev` for watch mode.
- `yarn build` for production build.
- `yarn clean` to clear dist/packages.
- `yarn lint` and `yarn format` for ESLint/Prettier.
- `yarn check` to run lint+build.
- `yarn package` for browser ZIPs, `yarn dist` for full pipeline.
