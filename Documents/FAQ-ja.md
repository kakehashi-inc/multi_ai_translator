# よくある質問（FAQ）

## 一般的な質問

### Multi-AI Translatorとは何ですか？

Multi-AI Translatorは、複数のAIプロバイダー（OpenAI、Claude、Gemini、Ollamaなど）を使用してウェブページを翻訳できるブラウザ拡張機能です。

### どのブラウザに対応していますか？

現在、以下のブラウザに対応しています：
- Google Chrome
- Microsoft Edge
- Chromiumベースのブラウザ（Brave、Operaなど）

Firefox版は将来的にリリース予定です。

### 無料で使用できますか？

拡張機能自体は無料ですが、AIプロバイダーのAPIを使用するには料金が発生する場合があります：

- **有料**：OpenAI、Claude、Gemini（従量課金）
- **無料**：Ollama（ローカルで動作）

Ollamaを使用すれば、完全無料で翻訳できます。

### APIキーは安全ですか？

はい、安全です。APIキーは以下のように保護されています：
- ブラウザのローカルストレージに暗号化されて保存
- 外部サーバーには送信されない
- 選択したAIプロバイダーにのみ送信

## インストールと設定

### インストール方法は？

1. Chrome ウェブストアまたはEdge アドオンストアから「Multi-AI Translator」を検索
2. 「追加」ボタンをクリック
3. 確認ダイアログで「拡張機能を追加」をクリック

詳細は[ユーザーガイド](USER_GUIDE-ja.md#インストール)を参照してください。

### APIキーの取得方法は？

各プロバイダーのAPIキー取得方法：

**OpenAI**：
1. [platform.openai.com](https://platform.openai.com/)にアクセス
2. アカウント作成またはログイン
3. 「API Keys」→「Create new secret key」

**Claude**：
1. [console.anthropic.com](https://console.anthropic.com/)にアクセス
2. アカウント作成またはログイン
3. 「API Keys」→「Create Key」

**Gemini**：
1. [makersuite.google.com/app/apikey](https://makersuite.google.com/app/apikey)にアクセス
2. Googleアカウントでログイン
3. 「Get API key」をクリック

### Ollamaの設定方法は？

1. [ollama.ai](https://ollama.ai/)からOllamaをダウンロード
2. インストール後、ターミナルで `ollama pull llama2` を実行
3. `ollama serve` でサーバーを起動
4. 拡張機能の設定で「Ollama」を選択
5. エンドポイント：`http://localhost:11434`
6. モデル：`llama2`（またはインストールしたモデル名）

### どのモデルを選べばいいですか？

用途別の推奨モデル：

**日常的な翻訳**：
- OpenAI: `gpt-3.5-turbo`
- Claude: `claude-3-haiku` または `claude-3-sonnet`

**高品質な翻訳**：
- OpenAI: `gpt-4-turbo-preview`
- Claude: `claude-3-opus`

**無料で使いたい**：
- Ollama: `llama2`、`mistral`、`gemma`

**専門用語が多い文書**：
- OpenAI: `gpt-4-turbo-preview`
- Claude: `claude-3-opus`

## 使用方法

### ページ全体を翻訳するには？

1. 翻訳したいページを開く
2. 拡張機能のアイコンをクリック
3. 「ページを翻訳」をクリック

### 選択したテキストだけを翻訳するには？

**方法1**：
1. テキストを選択
2. 右クリック→「選択テキストを翻訳」

**方法2**：
1. テキストを選択
2. `Ctrl+Shift+S`（Windows）または `Cmd+Shift+S`（Mac）

### 元の言語に戻すには？

1. 拡張機能のアイコンをクリック
2. 「元に戻す」ボタンをクリック

### 翻訳にどのくらい時間がかかりますか？

翻訳時間は以下の要因によって異なります：
- ページのサイズ
- 選択したモデル
- プロバイダーのレスポンス速度

**目安**：
- 小さなページ（〜500単語）：5-10秒
- 中規模ページ（〜2000単語）：20-40秒
- 大きなページ（〜5000単語）：1-2分

### 複数の言語に同時に翻訳できますか？

現在、一度に翻訳できるのは1つの言語のみです。別の言語に翻訳する場合は、設定でターゲット言語を変更してください。

## コストと制限

### 翻訳のコストはいくらですか？

コストは使用するプロバイダーとモデルによって異なります：

**OpenAI（2024年概算）**：
- GPT-3.5-turbo：$0.0005 / 1K トークン（入力）、$0.0015 / 1K トークン（出力）
- GPT-4-turbo：$0.01 / 1K トークン（入力）、$0.03 / 1K トークン（出力）

**Claude（2024年概算）**：
- Claude 3 Haiku：$0.00025 / 1K トークン（入力）、$0.00125 / 1K トークン（出力）
- Claude 3 Sonnet：$0.003 / 1K トークン（入力）、$0.015 / 1K トークン（出力）
- Claude 3 Opus：$0.015 / 1K トークン（入力）、$0.075 / 1K トークン（出力）

**Ollama**：完全無料（ローカル実行）

**参考**：1000単語の記事は約1300-1500トークン程度

### API制限はありますか？

はい、各プロバイダーには制限があります：

**OpenAI**：
- 無料アカウント：1分あたり3リクエスト
- 有料アカウント：プランによって異なる

**Claude**：
- プランによって異なる（通常、1分あたり数十リクエスト）

**Gemini**：
- 無料枠：1分あたり60リクエスト

**Ollama**：
- 制限なし（ローカル実行）

### 翻訳の品質はどうですか？

翻訳品質はモデルによって異なります：

**最高品質**：
- GPT-4、Claude 3 Opus：専門用語や複雑な文章に対応

**良好**：
- GPT-3.5、Claude 3 Sonnet：日常的な文章に十分

**実用的**：
- Claude 3 Haiku、Ollamaモデル：基本的な翻訳

## トラブルシューティング

### 翻訳が動作しません

以下を確認してください：
1. APIキーが正しく設定されているか
2. インターネット接続が有効か
3. プロバイダーのサービスが稼働しているか
4. API使用制限に達していないか

詳細は[トラブルシューティングガイド](TROUBLESHOOTING-ja.md)を参照してください。

### "API key invalid"エラーが表示されます

1. 設定ページでAPIキーを再確認
2. APIキーに余分なスペースや改行がないか確認
3. プロバイダーのダッシュボードでキーが有効か確認
4. 新しいキーを生成して試す

### 翻訳が途中で止まります

原因：
- ネットワークの問題
- API制限に達した
- ページが大きすぎる

解決策：
1. ページを再読み込みして再試行
2. 選択翻訳で部分的に翻訳
3. 設定でチャンクサイズを小さくする

### Ollamaに接続できません

1. Ollamaサーバーが起動しているか確認：`ollama serve`
2. エンドポイントが正しいか確認：`http://localhost:11434`
3. モデルがインストールされているか確認：`ollama list`
4. ファイアウォールがブロックしていないか確認

### 翻訳結果が文字化けしています

1. ブラウザのエンコーディング設定を確認
2. ページのHTMLエンコーディングを確認
3. 別のモデルを試す
4. ページを再読み込みして再試行

## プライバシーとセキュリティ

### データは収集されますか？

いいえ。Multi-AI Translatorは以下のデータを収集しません：
- 個人情報
- 翻訳内容
- 閲覧履歴

すべてのデータはブラウザのローカルストレージに保存されます。

### 翻訳内容はAIプロバイダーに送信されますか？

はい。翻訳するには、選択したAIプロバイダーにテキストを送信する必要があります。ただし：
- Multi-AI Translatorのサーバーには送信されません
- 直接プロバイダーのAPIに送信されます
- 各プロバイダーのプライバシーポリシーが適用されます

### 機密情報を翻訳しても安全ですか？

機密情報の場合は、Ollamaの使用を推奨します：
- すべての処理がローカルで完結
- データが外部に送信されない
- 完全なプライバシー保護

クラウドベースのプロバイダー（OpenAI、Claude、Gemini）を使用する場合は、各社のプライバシーポリシーを確認してください。
