## はじめに

Multi-AI Translatorは、複数のAIプロバイダーを使用してウェブページを翻訳できるブラウザ拡張機能です。OpenAI、Claude、Gemini、Ollamaなど、お好みのAIサービスを選択して使用できます。

### 主な機能

- **ページ全体の翻訳**：ワンクリックでウェブページ全体を翻訳
- **選択テキストの翻訳**：必要な部分だけを翻訳
- **複数のAIプロバイダー**：用途に応じて最適なAIを選択
- **元の内容に復元**：翻訳前の状態に簡単に戻せる
- **カスタマイズ可能**：翻訳設定を細かく調整

## インストール

### Chrome ウェブストアから（推奨）

1. [Chrome ウェブストア](https://chrome.google.com/webstore)にアクセス
2. "Multi-AI Translator" を検索
3. 「Chromeに追加」をクリック
4. 確認ダイアログで「拡張機能を追加」をクリック

### Microsoft Edge アドオンから

1. [Microsoft Edge アドオン](https://microsoftedge.microsoft.com/addons)にアクセス
2. "Multi-AI Translator" を検索
3. 「取得」をクリック
4. 確認ダイアログで「拡張機能の追加」をクリック

### 手動インストール（開発版）

開発版を使用する場合は、[Development-ja.md](Development-ja.md) を参照してください。

## 初期設定

インストール後、最初に使用するAIプロバイダーを設定する必要があります。

### 1. 設定ページを開く

拡張機能のアイコンをクリックし、「設定」ボタンをクリックします。

### 2. プロバイダーを選択

使用したいAIプロバイダーを選択します：

- **OpenAI**：最も人気のあるAIサービス（API料金が必要）
- **Claude**：Anthropic社のAI（API料金が必要）
- **Gemini**：Google社のAI（API料金が必要）
- **Ollama**：ローカルで動作するAI（無料、ただし自分でセットアップが必要）
- **OpenAI互換API**：LM Studioなどのローカルサービス

### 3. APIキーを設定

選択したプロバイダーのAPIキーを入力します。

#### OpenAI APIキーの取得方法

1. [OpenAI Platform](https://platform.openai.com/)にアクセス
2. アカウントを作成またはログイン
3. 「API Keys」セクションに移動
4. 「Create new secret key」をクリック
5. 生成されたキーをコピーして、設定画面に貼り付け

#### Claude APIキーの取得方法

1. [Anthropic Console](https://console.anthropic.com/)にアクセス
2. アカウントを作成またはログイン
3. 「API Keys」に移動
4. 「Create Key」をクリック
5. 生成されたキーをコピーして、設定画面に貼り付け

#### Gemini APIキーの取得方法

1. [Google AI Studio](https://makersuite.google.com/app/apikey)にアクセス
2. Googleアカウントでログイン
3. 「Get API key」をクリック
4. 生成されたキーをコピーして、設定画面に貼り付け

### 4. モデルを選択

各プロバイダーには複数のモデルが用意されています。用途に応じて選択してください：

- **高性能モデル**（GPT-4、Claude Opus）：複雑な文章、高精度が必要な場合
- **バランス型モデル**（GPT-3.5、Claude Sonnet）：日常的な使用に最適
- **高速モデル**（Claude Haiku）：素早い翻訳が必要な場合

### 5. ターゲット言語を設定

翻訳先の言語を選択します（例：日本語、英語、中国語など）。

## 基本的な使い方

### ページ全体を翻訳

1. 翻訳したいウェブページを開く
2. 拡張機能のアイコンをクリック
3. 「ページを翻訳」ボタンをクリック
4. 翻訳が完了するまで待つ（ページサイズによって時間が異なります）

**ヒント**：翻訳中は、ページを離れないでください。

### 選択したテキストを翻訳

#### 方法1：右クリックメニュー

1. 翻訳したいテキストを選択
2. 右クリックして「選択テキストを翻訳」を選択
3. ポップアップで翻訳結果を確認

#### 方法2：キーボードショートカット

1. 翻訳したいテキストを選択
2. `Ctrl+Shift+S`（Windows/Linux）または `Cmd+Shift+S`（Mac）を押す
3. ポップアップで翻訳結果を確認

### 元の内容に戻す

翻訳されたページを元の言語に戻すには：

1. 拡張機能のアイコンをクリック
2. 「元に戻す」ボタンをクリック

## 各AIプロバイダーの設定

### OpenAI

**推奨モデル**：
- `gpt-4-turbo-preview`：最高品質の翻訳
- `gpt-3.5-turbo`：コストパフォーマンスが良い

**設定項目**：
- APIキー：必須
- モデル：必須
- 温度：0.3（翻訳の一貫性を保つため低めに設定）

### Claude

**推奨モデル**：
- `claude-3-opus`：最高品質
- `claude-3-sonnet`：バランスが良い
- `claude-3-haiku`：高速・低コスト

**設定項目**：
- APIキー：必須
- モデル：必須
- 最大トークン数：4096（デフォルト）

### Gemini

**推奨モデル**：
- `gemini-pro`：汎用的な翻訳に最適

**設定項目**：
- APIキー：必須
- モデル：必須

### Ollama（ローカル）

**前提条件**：
1. [Ollama](https://ollama.ai/)をダウンロードしてインストール
2. モデルをダウンロード（例：`ollama pull llama2`）
3. Ollamaサーバーを起動

**推奨モデル**：
- `llama2`：汎用的
- `mistral`：高性能
- `gemma`：軽量

**設定項目**：
- エンドポイント：`http://localhost:11434`（デフォルト）
- モデル：インストールしたモデル名

### OpenAI互換API

LM Studio、LocalAI、その他のOpenAI互換APIを使用する場合：

**設定項目**：
- エンドポイント：サービスのURL（例：`http://localhost:1234/v1`）
- APIキー：必要に応じて
- モデル：サービスで利用可能なモデル名

## 高度な機能

### カスタムプロンプト

設定ページの「高度な設定」セクションで、翻訳プロンプトをカスタマイズできます。

デフォルトプロンプト：
```
以下のテキストを{target_language}に翻訳してください。
自然で読みやすい翻訳を心がけてください。

{text}
```

### バッチ翻訳

大きなページを翻訳する際、テキストは自動的に適切なサイズに分割されます。

設定可能な項目：
- チャンクサイズ：1回の翻訳リクエストで処理するテキスト量
- リクエスト間隔：API制限を回避するための遅延時間

### 翻訳キャッシュ

同じテキストの再翻訳を避けるため、翻訳結果はローカルにキャッシュされます。

**キャッシュをクリア**：
設定ページの「キャッシュをクリア」ボタンをクリック

## ヒントとコツ

### コスト削減

1. **適切なモデルを選択**：
   - 日常的な翻訳には、GPT-3.5やClaude Haikuで十分
   - 重要な文書にはGPT-4やClaude Opusを使用

2. **Ollamaを活用**：
   - ローカルで動作するため、API料金が不要
   - プライバシーも保護される

3. **選択翻訳を活用**：
   - ページ全体ではなく、必要な部分だけを翻訳

### 翻訳品質の向上

1. **適切なモデルを選択**：
   - 専門用語が多い場合は、高性能モデルを使用

2. **元言語を指定**：
   - 設定で元言語を指定すると、翻訳精度が向上

3. **カスタムプロンプト**：
   - 特定の用語や文体を指定してプロンプトをカスタマイズ

### プライバシー保護

1. **Ollamaを使用**：
   - すべての処理がローカルで完結
   - データが外部に送信されない

2. **APIキーの管理**：
   - APIキーは安全に保管
   - 定期的にキーをローテーション

3. **機密情報**：
   - 機密文書の翻訳には、ローカルモデル（Ollama）を推奨
